Simulasi Hash SHA-256

Panjang Hash: 0 karakter

Pembelajaran & Cara Menggunakan

Cara Menggunakan

  1. Input Data: Ketik teks apa saja ke dalam kolom "Input Teks".
  2. Observasi: Perhatikan kolom hasil Hash. Setiap perubahan satu huruf saja akan mengubah seluruh karakter hash.
  3. Panjang Tetap: Lihat indikator panjang karakter. Hash SHA-256 selalu menghasilkan 64 karakter heksadesimal, berapapun panjang inputnya.

Konsep Teknis: SHA-256

SHA-256 (Secure Hash Algorithm 256-bit) adalah fungsi matematika satu arah yang menjadi fondasi keamanan Blockchain.

  • Deterministik: Input yang sama pasti menghasilkan output yang sama.
  • Avalanche Effect: Perubahan kecil pada input menyebabkan perubahan drastis pada output (tidak bisa ditebak).
  • Pre-image Resistance: Mustahil mengembalikan Hash menjadi data aslinya.
